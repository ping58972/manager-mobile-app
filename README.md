- More detail... see package.json

## Getting Started

### Prerequisites

Nodejs with npm - lastest version.

```
Nodejs 12.6.0
```

### Installing

Download or clone the repository, then

```
npm install
```

### run the app

For Android:

```
react-native run-android
```

For ios:

```
react-native run-ios
```
